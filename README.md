## 🎰 Sobre o Projeto

O **PigJackpot** é um sistema web desenvolvido em Java que simula um cassino online, permitindo:

- Cadastro de usuários
- Gerenciamento de saldo (carteira)
- Realização de apostas
- Controle de partidas
- Depósitos e saques
- Histórico de transações

---

## 🧠 Conceitos Aplicados

- CRUD completo (Create, Read, Update, Delete)
- Relacionamento entre tabelas (1:N)
- Uso de chave estrangeira (FK)
- Modelagem de banco de dados (DER)
- Sistema com entidades dependentes (ex: Depósito + Itens)

---

## 💻 Tecnologias Utilizadas

- Java (Servlet/JSP)
- MySQL
- HTML / CSS
- JDBC
